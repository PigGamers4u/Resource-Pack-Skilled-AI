# Source Packs

Drop your own real, working resource packs here (or a subset of them —
manifest + the specific files you want mined as examples). These act as
ground-truth references the skills prefer over generating JSON from a
blank template.

Suggested layout:

```
references/source-packs/<pack-name>/
  manifest.json
  textures/...
  entity/...
  ...
```

Nothing is included by default — this repo ships as a scaffold, not with
someone else's pack bundled in.
